# Example Selection

Curated local examples are the first formatting references because they are small, stable, and selected for property-definition authoring.

Use `external/OPTIMADE` for standard property-definition analogies when available. Use `external/optimade-property-tools` for processing and provider workflow conventions when available.

Prefer examples with the same value shape, unit pattern, and entrytype context. For example, use scalar physical quantity examples for scalar quantities with units, list examples for vectors or per-site arrays, and dictionary examples for mappings.

Do not overfit to an example with different semantics. If several examples conflict, prefer the one closest in semantics and most consistent with current OPTIMADE source style.
